# TELUS CHR - Lab Result Status Cleanup

Makes lab result status indicators (e.g. "F" for Final) smaller and less prominent. Hides duplicate status indicators on multi-row results, showing only the first.

## Changelog

### v1.3.0
- Renamed file from chr- to telus-chr- prefix

### v1.2.0
- Adjusted status indicator width for better appearance

### v1.1.0
- Made status indicator narrower by overriding fixed width

### v1.0.0
- Initial release
- Smaller, subtler status indicators (reduced font size and opacity)
- Hide duplicate status on multi-row results (only first row shows status)
